Manually

You can integrate Yamasee into your project manually.

XCFramework

Use bash script to generate a universal (iOS Simulator and iOS device supported) Yamasee.xcframework.
In a project root folder run in Terminal

bash ./build.sh

Find framework at ./build/Yamasee.xcframework.

Copy Yamasee.xcframework file to the target project folder. Drag xcframewok file into Framework, Libraries, and Embedded Content section in Project -> General -> Target tab. It should be set as Embed & Sign.

import Yamasee in code where needed.

Done.

Structure

Auth Manager

  • Login and save tokens / refresh
  • manage Delivery tokens
  • logout

Data Manager

  • timer for
    • poll server initial / update
    • get delivery token from Auth Manager
    • publish turb / traffic new data
    • save locally in local storage
    • on init if logged in / on login serve locally saved data

Turb Manager

  • Store new turb data in memory
  • return turb data by alt
  • publish new turb data

Traffic Manager

  • store traffic data in memory
  • return traffic data
  • publish new traffic data
