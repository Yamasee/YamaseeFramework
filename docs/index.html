Installation

Manually

You can integrate Yamasee into your project manually.

XCFramework

Copy Yamasee.xcframework file to the target project folder. Drag xcframewok file into Framework, Libraries, and Embedded Content section in Project -> General -> Target tab. It should be set as Embed & Sign.

import Yamasee in code where needed.

Done.
